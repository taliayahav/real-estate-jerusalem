# Jerusalem Real Estate Platform

A property discovery platform for Jerusalem, similar to StreetEasy.

## What it does

- Browse and search property listings in Jerusalem
- Filter by price, bedrooms, neighborhood, and property type
- Natural language search powered by AI (e.g. "quiet apartment near parks with balcony")
- Agents can create and manage listings
- Users can save favorite properties

## Tech Stack

- **Backend:** Java, Spring Boot
- **Frontend:** React
- **Database:** PostgreSQL + pgvector (for semantic search)
- **AI:** OpenAI / Claude API (embeddings + natural language search)
